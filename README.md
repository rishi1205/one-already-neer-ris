# 💛 Neer & Rishi — One Year Anniversary Scrapbook

A digital scrapbook built with HTML, CSS & JavaScript. No frameworks, no dependencies — works entirely in the browser.

## 📁 File Structure

```
anniversary-scrapbook/
├── index.html          ← Main scrapbook file
├── css/
│   └── style.css       ← All styles & theme
├── js/
│   └── scrapbook.js    ← Navigation & interactions
└── README.md
```

## 🚀 Deploy to GitHub Pages

1. Create a new repository on GitHub (e.g. `neer-and-rishi`)
2. Upload all files maintaining the folder structure
3. Go to **Settings → Pages → Source → main branch / root**
4. Your scrapbook will be live at `https://yourusername.github.io/neer-and-rishi`

## 📖 Navigation

- **Click** Next / Prev buttons
- **Arrow keys** ← → to flip pages
- **Swipe** left/right on mobile

## 📷 Adding Photos

Click any photo placeholder to upload an image from your device.
